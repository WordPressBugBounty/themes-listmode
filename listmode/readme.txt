== Description ==
ListMode is a beautiful, responsive, minimalist WordPress theme for bloggers who want to keep their content clean. It has Customizer theme options, featured posts slider, sticky header and sticky sidebars, header navigation menu, header image support, full-width templates for posts and pages, 4 columns minimal post summaries, right sidebar, four columns footer, 60+ social icons, RTL support, translation support, scroll-to-top button, and more. ListMode PRO version has color options, font options, secondary menu, news ticker, 1-5 column post summaries, 4 thumbnail sizes for post summaries, featured posts widgets, tabbed widget, 10+ layouts for singular/non-singular pages, 10+ custom page/post templates, a settings panel for each post/page, separate widths/layout style/header style/footer style options for each post/page, an about and social widget, unique header images for each post/page, 3 header styles, 1-6 column footer, 25+ share buttons, post likes and post views, related posts, an infinite scroll or load more feature, more widget areas, and many other features. Free version demo: https://demo.themesdna.com/listmode/

Theme Link: https://themesdna.com/listmode-wordpress-theme/


== Frequently Asked Questions ==

= How to install the theme =

1.Login in to your WordPress dashboard.
2.Go to "Appearance" -> "Theme" and click on "Add New" button.
3.Click on "Upload Theme" button and choose the theme's zip file you have downloaded.
4.Click on "Install Now" button.
5.Activate the theme.
6.Now go to "Appearance" -> "Customize" to configure theme options.

Alternatively,

1.Login in to your WordPress dashboard.
2.Go to "Appearance" -> "Theme" and click on "Add New" button.
3.Search for "ListMode".
4.Click on Install and Activate.

If you need any help, or assistance we'd be happy to help. My email addresss: support@themesdna.com

= How to Regenerate Thumbnails =

"Regenerate Thumbnails" plugin helps you to regenerate your thumbnails to match with this theme.
1.Install and activate the plugin.
2.From the left hand navigation menu, click Tools > Regenerate Thumbnails.
3.On the next screen, click on "Regenerate Thumbnails For All X Attachments" or "Regenerate Thumbnails For The Y Featured Images Only" button.

= How to Add a Numbered Posts Navigation =

This theme uses "WP-PageNavi" plugin to display numbered page navigation of this theme. Just install and activate the plugin.
